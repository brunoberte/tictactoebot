#TicTacToe Bot

![Screenshot](https://raw.githubusercontent.com/brunoberte/tictactoebot/master/screenshot.png)

##Prerequisites
- PHP 7.1
- Composer https://getcomposer.org/

##Installation
```
composer install
```

##Running
```
composer run server
```
Open http://localhost:3080


### Running tests
```
composer run test
```
